# 📚 Python Functions Mastery

## 📌 About

Professional Python practice repository covering functions, parameters, return values, and problem-solving techniques for intermediate Python learners

---

## ✨ Features

- 📖 Function fundamentals (definition, parameters, return values)
- 🔧 Advanced function concepts (default arguments, *args, **kwargs)
- 🎯 Real-world function examples
- 💪 Hands-on practice problems

---

## 📂 Project Structure

| File | Description |
|---|---|
| `basic_functions.py` | Function definition and usage |
| `parameters_returns.py` | Parameters, return values, scope |
| `advanced_functions.py` | Default args, *args, **kwargs |
| `practice_problems.py` | Real-world function challenges |

## 💡 Key Concepts Covered

- ✅ Defining and calling functions
- ✅ Parameters and arguments
- ✅ Return values and multiple returns
- ✅ Variable scope (local vs global)
- ✅ Default parameters
- ✅ *args for variable arguments
- ✅ **kwargs for keyword arguments
- ✅ Function documentation (docstrings)
- ✅ Lambda functions
- ✅ Decorators introduction

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/ayeshamumtaz1057/python-functions-mastery.git
cd python-functions-mastery
```

### 2. Run the examples
```bash
python basic_functions.py
python parameters_returns.py
python advanced_functions.py
```

### 3. Try the practice problems
```bash
python practice_problems.py
```
